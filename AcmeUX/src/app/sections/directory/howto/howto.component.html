Directory Management How To

ACME Directory Management allows for the searching and adding new people to the directory.

Searching:

To search for individuals, type part of either their first or last name in to the search bar. The search performs dynamically and will display the results of individuals found as you type.

Spaces within the search term will behave as distinct separate searches, therefore a search for 'george burns' will return all results of people with the name 'George' and those with the name 'Burns'

Adding New People:

To add a new person, click the green plus button to the right of the search bar. This will allow you to provide information about the new person you wish to add.
